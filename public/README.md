# Public Assets Folder

## PDF Book Placement

Place your book PDF here:

```
/public/ai_agents_at_work.pdf
```

This file will be accessible at: `http://localhost:3002/ai_agents_at_work.pdf`

The book viewer page at `/book` will automatically load and display this PDF as an interactive carousel.

## Instructions

1. Copy your `ai_agents_at_work.pdf` file to this folder
2. Refresh the application
3. Navigate to the "Book" page or click "Learn More About the Book"
4. The PDF will display with page navigation, zoom, and download options

## Supported Formats

- PDF files only
- Recommended: Optimized PDFs for web viewing
- File size: Keep under 50MB for best performance
